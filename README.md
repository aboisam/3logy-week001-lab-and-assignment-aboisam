## 🚀 GETTING STARTED

### Step 1: Accept the Assignment
1. Click the GitHub Classroom invitation link (provided in Discord)
2. Accept the assignment
3. GitHub will create your personal repository: `week-01-labs-tickets-YOUR-USERNAME`

### Step 2: Clone Your Repository
```bash
# Navigate to where you want to store your projects
cd Desktop/Bootcamp

# Clone your personal repo (replace YOUR-USERNAME)
git clone https://github.com/3LOGY-Bootcamp-2026/week-01-labs-tickets-YOUR-USERNAME.git

# Navigate into the folder
cd week-01-labs-tickets-YOUR-USERNAME
```

### Step 3: Complete the Assignments
For each lab/ticket:
1. **Read the instructions:** Open `LAB-XX/instructions.md` or `TICKET-XX/requirements.md`
2. **Write your code:** Create your `.js` file inside the `code/` folder
3. **Test your code:** Run it in VS Code or browser console
4. **Check submission requirements:** Review the `submission.md` checklist

## 🌟 PRO TIPS

- **Start early:** Don't wait until the last minute
- **Read first, code second:** Understand requirements before coding
- **Comment as you go:** Explain your thinking while fresh
- **Use meaningful names:** `totalPrice` not `x`
- **Test edge cases:** What if input is 0? Negative? Empty?
- **Review demos:** Lecture demos show proper patterns
- **Collaborate:** Discuss logic, but write your own code
